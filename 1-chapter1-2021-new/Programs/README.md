#### 第6题 使用turtle绘图

本题中编写了两个辅助函数分别为*draw_ploygon*和*draw_angle*

- draw_ploygan(n, distance): 用于绘制正n边形，参数n表示边数，distance表示顶点到中心距离
- draw_angle(n, distance): 用于绘制正n角形，参数n表示角数，distance表示顶点到中心距离
